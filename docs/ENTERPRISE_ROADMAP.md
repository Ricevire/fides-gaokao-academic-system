# 企业级演进路线

## 已完成的生产基础

- 环境变量集中校验，启动时发现缺失或弱配置。
- 安全头、CORS 白名单、API 限流、登录限流。
- JSON 结构化日志与 `x-request-id` 请求追踪。
- `/health/live` 与 `/health/ready` 健康检查。
- 统一错误响应，屏蔽生产环境内部错误细节。
- 请求体基础校验，覆盖核心写接口。
- Dockerfile、docker-compose 与部署文档。
- 优雅关闭，退出前释放 HTTP 服务和数据库连接池。
- 数据库迁移体系：`migrations/`、`schema_migrations` 版本表、迁移状态、基线化和回滚命令。
- 生产环境禁止默认管理员初始密码。
- 管理员可生成或重置教师、学生账号，账号首次登录必须改密。
- 审计日志：登录、失败登录、改密、账号发放、敏感数据增删改、权限拒绝。
- 审计日志接入：JSON 结构化输出、可选 SIEM Webhook、异常 IP 登录/高频失败登录/账号重置高频告警。
- 权限矩阵：管理员、教务、班主任、教师、学生分层授权到接口、页面和核心操作按钮。
- API 自动化测试：覆盖认证、选科、学生、课表、成绩核心链路，并使用隔离测试库。
- 备份恢复流程：逻辑备份、指定库恢复、恢复演练报告；已完成一次恢复演练并通过行数校验。
- 迁移合规检查：新增数据库结构变更必须通过 `migrations/` 版本文件进入版本管理，并检查命名、导出、checksum、未应用迁移和孤儿迁移。
- Excel 导入导出和批量校验：教师、学生档案支持 `.xlsx` 模板、导出、预检和按工号/学号幂等导入。
- 分页、筛选、排序：教师、学生、行政班、教学班、考试、成绩列表支持分页响应和前端分页器，避免大数据量一次性加载。
- 操作日志检索、告警分级处置和慢接口监控：审计日志/告警支持分页检索、多条件过滤、告警关闭/确认/降级处置，慢接口自动写入 `slow_api` 告警。
- 前端工程化构建：新增 `frontend/src` TypeScript 源码目录、内部 UI 组件库、esbuild 构建脚本，并在 Docker 构建阶段生成前端产物。
- 自动排课、冲突检测、教师课时统计和晚自习课表：按课程计划自动生成教学班课表，手工排课前检测教师/教室/班级/学生冲突，并按教师汇总普通课与晚自习课时。
- 多校区、多学年、多租户数据隔离：新增租户、校区、学年基础表，登录凭证携带组织上下文，教师/学生/班级/教学班/课表/考试/审计等核心接口按租户隔离，并支持校区和学年筛选。
- 统一身份认证和短信通知：接入 OIDC 授权码统一登录，支持外部身份与本地账号绑定；新增短信模板、发送记录、通用 Webhook 网关和账号发放短信提醒。
- 数据看板、成绩趋势分析、选科组合预测：工作台展示成绩概览、分数段、学科均分和趋势，选科预测输出预测人数、建议教学班、置信度和风险等级。
- 灾备部署、灰度发布、回滚与容量压测：新增发布运行手册、Nginx 蓝绿灰度模板、发布/回滚门禁检查脚本和无依赖容量压测脚本。
- 教师多职务体系：教师可同时担任段长、副段长、年级学科负责人、班主任和任课老师，班主任/任课关系同步行政班与教学班。
